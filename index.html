색상 혼합 규칙

2색 혼합

빨강 + 노랑 = 주황
빨강 + 파랑 = 보라
노랑 + 파랑 = 초록
빨강 + 흰색 = 분홍
노랑 + 흰색 = 레몬
파랑 + 흰색 = 하늘색

3색 혼합

빨강 + 노랑 + 흰색 = 연주황
빨강 + 파랑 + 흰색 = 연보라
노랑 + 파랑 + 흰색 = 연초록
빨강 + 노랑 + 파랑 = 검정

특수 혼합

검정 + 흰색 = 회색
